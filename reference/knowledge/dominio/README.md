# Conocimiento de Dominio — TMS

> **Capa:** Evidencia de Producto Aplicado
> **Propósito:** Documentación del dominio de gestión de transportes de Unimar

## Áreas de Conocimiento

- Procesos operativos de transporte de carga
- Integración con depósitos temporales y aduanas
- Gestión de flota y conductores
- Facturación de fletes y servicios logísticos
- Tracking y telemetría de unidades
- Documentación regulatoria (manifiestos, BL, DUA)

Este directorio contiene documentación detallada del dominio, investigaciones, análisis de procesos y referencias del negocio de Unimar.
