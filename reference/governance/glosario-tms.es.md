## Términos del Dominio TMS

| Término | Definición |
| :------ | :--------- |
| TMS (Transportation Management System) | Sistema de Gestión de Transportes; plataforma para planificar, ejecutar y optimizar el movimiento físico de cargas |
| Flete | Servicio de transporte de carga contratado por un cliente |
| Unidad de Transporte | Vehículo (camión, tráiler, cisterna) utilizado para el transporte |
| Conductor | Persona que opera la unidad de transporte |
| Ruta | Trayecto planificado para una operación de transporte |
| Manifiesto | Documento que registra la carga transportada en una unidad |
| Tracking | Seguimiento en tiempo real de la posición de una unidad |
| Bill of Lading (BL) | Conocimiento de embarque; documento legal del contrato de transporte |
| Depósito Temporal | Almacén autorizado para mercancías en tránsito aduanero |
| Puerto de Llegada | Puerto de destino de la carga |
| Puerto de Embarque | Puerto de origen de la carga |
| Contenedor Vacío | Contenedor sin carga; su gestión logística es crítica |

## Excepciones

Los siguientes términos se conservan en inglés por ser acrónimos o estándares internacionales: TMS, BL, GPS, ETA, ETD, POD.
