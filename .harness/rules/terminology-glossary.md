# Glosario de Terminología — unimar_tms

> **Herencia:** Adopta el [Glosario de Terminología de unimar_arch](https://github.com/mhernandez-unimar/unimar_arch/blob/main/.harness/rules/terminology-glossary.md) y lo extiende con términos del dominio TMS.

## Términos Técnicos (heredados)

| Término | Definición | Notas |
| :------ | :--------- | :---- |
| Architecture Decision Record (ADR) | Registro formal de una decisión arquitectónica | Acrónimo conservado |
| Bounded Context | Unidad de modelo con límites explícitos (DDD) | Término DDD |
| Domain-Driven Design (DDD) | Enfoque de diseño orientado al dominio | Acrónimo conservado |
| Operación de Herencia (Adopt/Extend/Override/N/A) | Etiqueta de registro para triaje local de patrones | Verbos en inglés por consistencia |
| Quality Gate | Criterio objetivo de paso para una fase o release | Conservado verbatim |
| SDLC | Software Development Life Cycle | Acrónimo conservado |

## Términos del Dominio TMS

| Término | Definición |
| :------ | :--------- |
| TMS (Transportation Management System) | Sistema de Gestión de Transportes |
| Flete | Servicio de transporte de carga contratado por un cliente |
| Unidad de Transporte | Vehículo utilizado para el transporte de carga |
| Conductor | Persona que opera la unidad de transporte |
| Ruta | Trayecto planificado para una operación de transporte |
| Manifiesto | Documento que registra la carga transportada en una unidad |
| Tracking | Seguimiento en tiempo real de la posición de una unidad |
| Depósito Temporal | Almacén autorizado para mercancías en tránsito aduanero |
| Contenedor Vacío | Contenedor sin carga; su gestión logística es crítica |

## Excepciones

Términos conservados en inglés: TMS, BL, GPS, ETA, ETD, POD, ADR, API, CI, CLI, DDD, SDLC.
